# Data Directory

Place your department documents in the appropriate subfolder:

- `general/`     → Company-wide docs (accessible by all roles)
- `finance/`     → Financial reports, budgets (finance_analyst, executive)
- `engineering/` → Architecture docs, API specs (engineer, executive)
- `marketing/`   → Campaign reports, brand guides (marketing_specialist, executive)
- `hr/`          → Policies, benefits guides (hr_representative, executive)

Supported formats: `.pdf`, `.docx`, `.md`, `.csv`, `.pptx`
